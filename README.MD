# Pure dark

Pure black & white theme

## Screenshot

![pure dark](screenshot.png "pure dark")

## Development

* go to chrome://extensions
* enable "Developers mode"
* click to "Pack extension..."
* select root "pure-dark/1.0_0/"
* pack
* drop file 1.0_0.crx to new tab in browser
